# TICKET-050: 【Web】シフト画面の改善

## 担当
Frontend

## 概要
マネージャー向けシフト管理画面（/shifts, /shifts/create）を使いやすく改善する。

## リポジトリ
`ikki-dali/airport-shift`

## 対象画面

### 1. シフト一覧画面（/shifts）
現状：テーブル形式でシフト一覧表示

改善案：
- カレンダービュー追加（月間カレンダーでシフト確認）
- 人手不足の日を赤ハイライト（ダッシュボードと同様）
- フィルターのUI改善（コンパクトに）

### 2. シフト作成画面（/shifts/create）
現状：週単位でシフト作成

改善案：
- より直感的なUI
- ドラッグ&ドロップでシフト追加
- 一括作成機能の改善

## 優先度
1. シフト一覧のカレンダービュー
2. 人手不足の赤ハイライト
3. UIの全体的な改善

## 完了条件
- [ ] シフト一覧にカレンダービューがある
- [ ] 人手不足の日が赤ハイライト
- [ ] 見やすく使いやすいUI
- [ ] ビルドが通る

## 備考
- まずはカレンダービュー追加を優先
- 大きな改修より、見やすさ・使いやすさ重視
