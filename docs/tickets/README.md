## 📝 チケットの読み方

各チケットには以下の情報が含まれています:

- **ステータス**: 進捗状況（完了/進行中/未着手）
- **優先度**: ⭐の数（5つが最高）
- **複雑度**: Simple/Medium/Complex
- **概要**: チケットの目的
- **成果物**: 実装すべき内容
- **依存関係**: 他のチケットとの依存関係
- **機能要件**: 詳細な仕様
- **UI設計**: 画面イメージ
- **実装例**: コード例
- **テスト項目**: 確認すべき項目
- **完了条件**: 完了の基準
- **見積もり工数**: 必要時間
- **開始/完了予定日**: スケジュール
