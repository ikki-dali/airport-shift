# TICKET-046: ダッシュボード縦スクロールレイアウトに変更

## 担当
Frontend

## 概要
タブ切り替えから縦スクロールレイアウトに変更する。
画面が広いのでタブ切り替えは不要、一覧性を高める。

## リポジトリ
`ikki-dali/airport-shift`

## 変更内容

### Before（タブ切り替え）
```
[Today] [This Week] [This Month]
      ↓ 選択した1つだけ表示
```

### After（縦スクロール）
```
── 統計バー ────────────
充足率 | 人数不足 | 承認待ち

── Today ──────────────
今日の出勤者リスト

── This Week ──────────
今週のシフトリスト
（空いてる日は赤ハイライト）

── This Month ─────────
カレンダー表示
（空いてる日は赤ハイライト）
```

## 実装メモ
- DashboardTabs.tsx を削除または改修
- TodayTab, WeekTab, MonthTab をそのまま縦に並べる
- 各セクションにタイトル（Today, This Week, This Month）を付ける
- スクロールで全部見れるように

## 完了条件
- [ ] タブ切り替えではなく縦スクロールになっている
- [ ] Today, This Week, This Month が全部表示される
- [ ] 各セクションにタイトルがある
- [ ] スマホでも見やすい（レスポンシブ）
- [ ] ビルドが通る
