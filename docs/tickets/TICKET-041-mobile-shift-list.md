# TICKET-041: 【Mobile】確定シフト一覧画面

## 担当
Frontend

## 概要
バイトが自分の確定シフトを見やすく確認できる画面を作成する。
**第1優先機能** - バイト側で最も重要な画面。

## リポジトリ
`ikki-dali/airport-shift-mobile`

## 画面要件

### ホームタブ（app/(tabs)/index.tsx）

#### ヘッダー
- 現在の年月を表示（例: 2024年2月）
- 月切り替えボタン（< >）

#### メインエリア
- **カレンダー形式**または**リスト形式**で確定シフトを表示
- 各シフトに表示する情報:
  - 日付（2/5 月）
  - 勤務地
  - 時間帯（09:00-18:00）
  - 勤務記号

#### 表示の工夫
- 今日のシフトは目立つようにハイライト
- 確定済みシフトのみ表示（予定は表示しない）
- シフトがない日は薄くグレーアウト

### データ取得
```typescript
// Supabaseから自分のシフトを取得
// 注: 今回は認証未実装なので、全シフトまたはダミーユーザーのシフトを表示
const { data } = await supabase
  .from('shifts')
  .select(`
    *,
    location:locations(*),
    duty_code:duty_codes(*)
  `)
  .eq('status', '確定')
  .gte('date', startOfMonth)
  .lte('date', endOfMonth)
  .order('date')
```
