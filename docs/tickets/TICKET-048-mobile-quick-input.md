# TICKET-048: 【Mobile】シフト希望の高速入力モード

## 担当
Frontend

## 概要
現状：1ヶ月分の希望を出すのに約60回タップ必要（日付30回 + ○△×選択30回）
目標：**10タップ以下**で1ヶ月分の希望を入力できるようにする

## リポジトリ
`ikki-dali/airport-shift-mobile`

## 改善案

### 1. デフォルト一括設定
最初に「基本の希望」を選ぶ：
- 「基本: ○（出勤可能）」← これを選ぶと全日が○になる
- 「基本: ×（出勤不可）」← これを選ぶと全日が×になる

その後、例外の日だけタップして変更。

**例: 基本○で、土日だけ×にしたい場合**
1. 「基本: ○」を選択（1タップ）
2. 土日をタップして×に変更（8タップ）
= **合計9タップ**

### 2. 曜日一括ボタン
- 「平日を○」「土日を×」ボタン
- 「月曜を×」「火曜を○」など曜日単位で一括設定

### 3. 範囲選択（ドラッグ）
- カレンダー上で長押し → ドラッグで複数日を選択
- 選択した日に一括で○△×を設定

### 4. よく使うパターン保存
- 「いつものパターン」を保存
- 次月以降はワンタップで適用

## 推奨実装

**Phase 1（今回やる）**
- デフォルト一括設定（案1）
- 曜日一括ボタン（案2）

**Phase 2（後で）**
- 範囲選択
- パターン保存
